****Compilation*****
Make sure you are in the src file folder of A1, then type make into the LXTerminal.
Run the program by typing "./maze"

*********Running the program*******
You will spawn in the maze, use the ijkl keys to move.
i will make you move up one character.
j will make you move left one character.
k will make you move down one character.
l will make you down right one character.

You will win when you get to the E on the far side of the maze wall.

*You may quit at anytime by pressing 'q'.

****Known Limitations******
pressing ctrl + c will close the program at any time.
